This project is a simple **Blog CRUD (Create, Read, Update, Delete) application** built using **Node.js**, **Express**, and **MongoDB Atlas**.  
The application provides a RESTful API and a minimal frontend interface for manual testing of all CRUD operations.

The project is designed for educational purposes and demonstrates backend fundamentals such as API design, database integration, validation, error handling, and basic frontend interaction.


## Features

 Full CRUD functionality for blog posts
 RESTful API architecture
 MongoDB Atlas cloud database
 Data validation using Mongoose
 Automatic timestamps (`createdAt`, `updatedAt`)
 Default values for optional fields
 Error handling for invalid requests
 Simple frontend UI for manual testing
 Clean project structure
